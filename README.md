# Ai-cartooniser
Artifice Nexus is an AI-driven web application that transforms real images into cartoon, sketch, and artistic oil-paint styles using deep-learning based image stylization. The platform provides a smooth, real-time experience with webcam capture, instant previews, and fast ML inference.

Features:
-> AI Cartoonization
    Converts photos into high-quality cartoon, sketch, and oil-painting styles using a CNN-based model.
-> Webcam Image Capture
    Capture images directly from the webcam for instant stylization.
-> Real-Time Preview
    Fast inference pipeline allowing users to see results instantly.
-> Full-Stack AI Integration
    React frontend communicates with a Flask ML backend for seamless processing.
-> Multiple Style Modes
    Switch between cartoon, sketch, oil-paint, and more artistic filters.

Tech Stack:
-> Machine Learning & Computer Vision
-> PyTorch
-> CNN-based White-Box Cartoonizer
-> OpenCV
-> Image Preprocessing & Normalization
-> Feature Extraction & Style Transfer

Backend:
-> Python
-> Flask (REST API)

Frontend:
-> React.js
-> Tailwind CSS
-> Webcam API
-> Axios
